Defining ShzPlus C++:
Using ShzPlus in C++ is a quite big work because it is quite difficult to those that are not friendly to C++. We recommend you to use a easy-to-use IDE Dev-C++ if you are quite new to C++ and an IDE Code::Blocks or Visual Studio for those that are very friendly to C++.
Download Dev-C++  or...  Code::Blocks  or...  Visual Studio
After you finished with those blasted IDE works, it is time to get started with ShzPlus!
First, you can start with a include preprocessor and a using namespace:
#include <shzplus.h>
using namespace shzplus;

Then you can make a brief ShzPlus Project:
#include <shzplus.h>
using namespace shzplus;
process {
     shzplus ( version ( 3));
    //Enter code here...
    return 0;
};
